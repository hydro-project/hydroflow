Struct hydroflow_plus::stream::Stream

source · 
pub struct Stream<'a, T, W, N: Location + Clone> { /* private fields */ }
Expand description

An infinite stream of elements of type T.

Type Parameters:

  • 'a: the lifetime of the final Hydroflow graph, which constraints which values can be captured in closures passed to operators
  • T: the type of elements in the stream
  • W: the windowing semantics of the stream, which is either Async or Windowed
  • N: the type of the node that the stream is materialized on
