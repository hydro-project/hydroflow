Replaces singleton references #my_var with the code needed to actually get the value inside.

  • tokens - The tokens to update singleton references within.
  • resolved_idents - The context StateHandle varnames that correspond 1:1 and in the same order as the singleton references within tokens (found in-order via preprocess_singletons).

Generates borrowing code (std::cell::RefCell::borrow_mut). Use postprocess_singletons_handles for just the StateHandles.
