Expand description

This extra layer is needed due to the ownership order. In the functional chaining syntax each operator owns the previous (can only go in order things are called/defined), but in the end we need each pusherator to own the next pusherator which it’s pushing to.

This is the already-reversed, PushSurface does the actual reversing.
